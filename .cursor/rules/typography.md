# Typography Rules

## Type Scale
- Use a modular scale (e.g., 1.25 ratio): 12, 14, 16, 20, 25, 31, 39, 49.
- Base body size 16px minimum on web.
- Limit to 2 typefaces (one display/heading, one body). One is often enough.

## Hierarchy
- Differentiate levels with size AND weight, not just size.
- Headings: tighter line-height (1.1–1.3). Body: looser (1.5–1.7).
- Headings can use negative letter-spacing; body text rarely needs tracking changes.

## Readability
- Line length 45–75 characters for body copy.
- Left-align body text; avoid justified text on the web.
- Use weight (500–700) for emphasis, not italics or all-caps for long strings.

## Pairing & Polish
- Pair a personality font for display with a neutral, highly legible font for body.
- Use proper typographic characters: curly quotes, em dashes, ellipsis.
- Set `font-feature-settings` for tabular numbers in tables/data.
