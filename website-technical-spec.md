# The Prompt Cheat Sheet

The gap is yours to close. This is how.

## Before you type, answer three things
- What already exists
- What you're trying to achieve
- What you've tried or ruled out

Can't answer all three? You're not ready yet.

## Build the prompt
- **Always:** Task. What you want done.
- **Usually:** Context, grounding, rules, skills, process.
- **When useful:** Role, constraints, examples, format.

Don't fill every lever. Order off the menu the task needs. Too little and it guesses, too much and it gets distracted.

## Set the two dials
- **Specificity:** how much you direct vs. leave open.
- **Depth:** how much thinking you ask for.

Don't set them and the AI sets them for you. It guesses average.

## When the output misses
- Task unclear -> restate with more specificity
- Context missing -> add the grounding you left out
- Depth wrong -> say so directly
- Format wrong -> show an example

Refine when it still understands you. Restart when it's locked onto the wrong idea.

## Make it check itself
Paste into any high-stakes prompt:

> Before I review this, check it yourself. Does it answer the original question? What assumptions did you make that I should verify? Where are you least confident?
